# LoL Comms Guide

英雄联盟语音沟通中英对照表，侧重语音开黑表达，而非聊天栏缩写。

## Features

- 中文 / 英文实时搜索
- 分类浏览
- 卡片 / 表格视图切换
- 一键复制英文表达
- 收藏常用词条
- 响应式布局
- LoL-inspired fantasy esports UI

## Tech Stack

- Vite
- React
- TypeScript
- GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
